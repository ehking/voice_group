# Dorehmi Analyzer (دورهمی آنالایزر)

MVP محلی برای پردازش مکالمه‌های چندنفره فارسی: تفکیک گوینده، متن محاوره‌ای، برچسب احساسی، تحلیل تعامل و خلاصه.

## اجرای سریع
```bash
sudo apt install ffmpeg
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
make dev
```

وب‌پنل فارسی در `http://localhost:8000/` در دسترس است. خروجی‌ها در پوشه `storage/` ذخیره می‌شوند.

## ساختار پوشه‌ها
- `backend/`: FastAPI، قالب‌ها، مدل داده
- `worker/`: پردازش صف و اجرای پایپ‌لاین
- `models/`: ماژول‌های پیش‌پردازش، دیاریزیشن، STT، احساس، تحلیل و خلاصه
- `frontend/`: فضای رزرو برای کدهای فرانت در آینده (الان قالب‌ها در backend)
- `docs/`: راه‌اندازی و توضیحات
- `tests/`: تست‌های واحد
- `storage/`: فایل‌های محلی (آپلود، لاگ، نتایج)

## وضعیت MVP
- پایپ‌لاین ساده قابل جایگزینی با مدل‌های دقیق‌تر
- صف محلی مبتنی بر SQLite، پشتیبانی چند کار موازی
- رابط RTL با فونت وزیرمتن و به‌روزرسانی خودکار وضعیت‌ها
